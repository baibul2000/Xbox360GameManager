Xbox 360 Game Manager — мощный инструмент с графическим интерфейсом для автоматизации процесса подготовки игр для Xbox 360. Программа скачивает образы игр с общедоступного репозитория, распаковывает их, конвертирует в формат GOD (Games on Demand) и при необходимости отправляет готовые файлы на консоль по FTP. Всё управление происходит через удобный интерфейс с несколькими вкладками и детальным логированием.

## 🚀 Возможности

- **Загрузка списка игр** с сайта `myrient.erista.me` (Redump Xbox 360).
- **Многопоточное скачивание** ZIP-архивов с поддержкой докачки (через aria2).
- **Распаковка** ZIP, RAR, 7z.
- **Конвертация ISO → GOD** через `iso2god-rs`.
- **Управление очередью**: пауза, остановка, повторная попытка.
- **Три вкладки**: «Все игры», «В процессе», «Скачанные».
- **Сохранение настроек** в `config.json`.
- **FTP-клиент** с проверкой подключения и настройкой режимов.
- **Трей-иконка** и информативные логи.
- **Кнопка пожертвования** для поддержки разработки.
- **Смена темы** (светлая/тёмная).

⚙️ Настройка
Перед началом работы укажите пути к трём папкам:

Папка загрузок – куда будут сохраняться ZIP-архивы.

Папка архивов – временная папка для распакованных ISO (можно потом удалить).

Папка для GOD – куда будут помещены готовые GOD-папки.

Эти пути можно изменить в любой момент через интерфейс; они сохранятся в config.json.

FTP-настройки
Если вы планируете отправлять игры на консоль по FTP, включите опцию «Передать FTP» и заполните поля:

IP/Порт – например, 192.168.0.53:2221

Пользователь, Пароль

Путь Консоли FTP – удалённая папка, куда будут загружаться игры (обычно /hdd1/content/00000000000).

Кнопка «Подключить» позволяет проверить соединение до начала отправки.

🎮 Использование
Главное окно
Обновить список – загружает актуальный перечень игр с сайта (выполняется автоматически при запуске).

Выбор игр – в таблице «Все игры» можно отметить галочками нужные игры.

Кнопки управления:

Начать – добавляет выбранные игры в очередь и запускает обработку.

Отменить – останавливает все текущие задачи (временные файлы не удаляются).

Пауза – приостанавливает выполнение текущих задач (можно продолжить позже).

Добавить игру – добавляет выбранные игры в очередь, не запуская воркеры (удобно для пакетного добавления).

Очистить временные – удаляет все ZIP-архивы и распакованные папки (GOD не трогает).

Удалить выбранные – удаляет файлы выбранных игр (работает на вкладках «В процессе» и «Скачанные»).

Повторить – добавляет в очередь все игры, у которых статус содержит «Ошибка».

Тема – переключение между светлой и тёмной темой.

ПОЖЕРТВОВАТЬ – открывает страницу для поддержки разработчика.

Вкладки
Все игры – полный список игр с сайта. Игры, уже имеющие ZIP или GOD, помечаются статусом «Скачано», «Скачано (распаковано)» или «Установлено» и автоматически скрываются из этой вкладки.

В процессе – игры, находящиеся в очереди, скачивании, распаковке, конвертации или отправке по FTP. Здесь также есть колонка выбора.

Скачанные – игры, у которых присутствует хотя бы один из файлов (ZIP, распакованный ISO или GOD). Можно выбрать и, например, удалить их или повторно добавить в очередь.

Ход выполнения
В колонке «Прогресс» отображается зелёная полоса с процентом выполнения.

В логе под таблицей выводятся подробные сообщения: начало операций, скорость скачивания, оставшееся время, завершение.

При свёртывании в трей иконка показывает количество активных задач и средний прогресс.
