[![Latest Release](https://img.shields.io/github/v/release/ваш-username/Xbox360GameManager)](https://github.com/ваш-username/Xbox360GameManager/releases)

**Xbox 360 Game Manager** – мощный инструмент с графическим интерфейсом для автоматизации процесса подготовки игр для Xbox 360. Программа скачивает образы игр с общедоступного репозитория, распаковывает их, конвертирует в формат GOD (Games on Demand) и при необходимости отправляет готовые файлы на консоль по FTP.

---

## 📥 Скачать программу

Готовые сборки доступны на странице [Releases](https://github.com/ваш-username/Xbox360GameManager/releases). Скачайте последний архив `Xbox360GameManager.zip`, распакуйте его и запустите `Xbox360GameManager.exe`.

> **Важно**: программа требует наличия внешних утилит `aria2c.exe` и `iso2god-rs.exe`. Они уже включены в архив релиза. Если вы собираете программу самостоятельно, скачайте их отдельно (ссылки ниже).

## 🚀 Возможности

- **Загрузка списка игр** с сайта `myrient.erista.me` (Redump Xbox 360).
- **Многопоточное скачивание** ZIP-архивов с поддержкой докачки (через aria2).
- **Распаковка** ZIP, RAR, 7z.
- **Конвертация ISO → GOD** через `iso2god-rs`.
- **Управление очередью**: пауза, остановка, повторная попытка.
- **Три вкладки**: «Все игры», «В процессе», «Скачанные».
- **Сохранение настроек** в `config.json`.
- **FTP-клиент** с проверкой подключения и настройкой режимов.
- **Трей-иконка** и информативные логи.
- **Кнопка пожертвования** для поддержки разработки.
- **Смена темы** (светлая/тёмная).

## 📋 Системные требования

- **ОС**: Windows 10/11 (64-bit)

- **Свободное место**: минимум 10 ГБ (для игр)

## ⚙️ Настройка и использование

1. **Распакуйте архив** в любую папку.
2. Запустите `Xbox360GameManager.exe`.
3. Укажите пути к папкам:
   - **Папка загрузок** – для ZIP-архивов.
   - **Папка архивов** – временная для распакованных ISO.
   - **Папка для GOD** – для готовых GOD-папок.
4. Если нужна отправка по FTP, включите опцию и заполните поля (IP, порт, логин, пароль, путь на консоли). Кнопка **«Подключить»** проверит соединение.
5. Нажмите **«Обновить список»**, чтобы загрузить список игр.
6. Выберите игры галочками и нажмите **«Начать»**. Программа скачает, распакует, сконвертирует и (опционально) отправит по FTP.
7. Следите за прогрессом в таблице и логе.

Подробнее об интерфейсе и кнопках – в разделе [Использование](#использование).
